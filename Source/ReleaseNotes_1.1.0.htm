Change Log

  • This is the first public release of the Engage: Booking module
  • Allows users to request a time slot on the calendar
  • Allows module editors to approve or deny that time slot
  • Sends notification emails when new appointments are requested
  • Sends notification emails when appointments are accepted or declined
  • Only shows appointment details to module editors
  • More details can be found on the Engage: Booking Change Log

Known Issues

  • Modules using different versions of the Telerik RadControls for ASP.NET AJAX (including many OnyakTech modules) may not be able to function alongside Engage: Booking.
